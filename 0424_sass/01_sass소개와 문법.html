SASS - sass(Syntactically Awesome StyleSheets)소개

CSS pre - processer(전처리기)로서 css반복을 줄여서 시간을 절약할 수 있도록 해주는 css언어의 확장언어.

변수 및 조건문 등의 새로운 문법을 통해 css를 구조화하여 작업할 수 있다.

코드가 지저분해질 수 있고, 코드가 많아질 수록 유지보수가 어려움.

프로그램 설치는 sass공식사이트에서 GUI,CLI방식의 프로그램들 중에서 자신이 편한 프로그램을 다운받아 설치한다.VS Code 플러그인 'sass live compiler를 설치하면 좀 더 편하게 사용 가능.'

sass파일의 확장자는 .sass, .scss 2가지가 존재하며 문법의 차이가 있고 거의 같다고 보면 된다.css문법과 거의 비슷하여 주로 scss파일을 사용하며, 브라우저에서는 css만 인식하기 때문에 .scss파일을 .css로 변환하는 작업을 거쳐야(컴파일을) 사용할 수 있다.
