01_sass_변수의 개념

변수 : sass에서도 변수개념을 활용하여 다양한 값을 저장할 수 있다.

문법 : $변수명:값;

변수에 들어갈 수있는 값은 '문자열','숫자','색상',불표현식(boolean),리스트, null값

변수의 범위

-전역변수 : 문서에 바로 작성한 변수로 모든요소에서 사용가능

- 지역변수 : 특정 코드블록({})에서 작성한 변수로 코드블록을 벗어나면 사용 불가능

    {
      $bgcolor01: #ccc;
      background-color: $bgcolor01;
      border:1px solid $bgcolor01;
    }
    선택자 안에서 같은 서식이있다면 여러번 적용가능.
