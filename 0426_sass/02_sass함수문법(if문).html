if문법
- 조건에 따른 결과를 true/false로 나눠서 처리하는 삼항연산자와 비슷한 문법입니다.
조건식이 true이면 '참', 그렇치 않으면'거짓'으로 반환합니다.
    ** 문법
    if(조건식, 참표현식, 거짓표현식);
    
    ** 스크립트
    조건식? 참표현식:거짓표현식;

2. @if문법

자바스크립트의 if문과 비슷하게 사용

@else를 붙여 false일 경우 실행하고, 다중 if문도 실행할 수 있다.

    ** 문법
    $@if(조건){
      //참표현식
    }

    @if(조건식){
      //참표현식
    }@else{
      //거짓표현식
    }

    @if(조건식1){
      //참표현식
    }@else if(조건식2){
      //거짓표현식
    }@else{
      //위 조건 모두 거짓일 경우 표현식
    }
